🖥️ Web Servers

A web server is software and hardware that stores, processes, and delivers web pages to users upon request.

How it works:

  1. User requests a webpage via browser
  2. Request travels to the web server
  3. Server processes the request
  4. Server sends back the requested files (HTML, CSS, images)
  5. Browser displays the page

Popular web servers: Apache, Nginx, Microsoft IIS.
