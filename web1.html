📟 Web 1.0 (The Read-Only Web)

Web 1.0 refers to the first stage of the World Wide Web, roughly from 1991 to 2004.

Characteristics of Web 1.0:

  • Read-only: Users could only read content, not interact with it
  • Static pages: Websites were simple HTML pages that rarely changed
  • No user interaction: No comments, likes, or user-generated content
  • Information delivery: Websites were like digital brochures
  • Slow connections: Dial-up internet was common

Examples of Web 1.0: Early Yahoo!, GeoCities, basic company websites, online encyclopedias.

Web 1.0 was about reading information - not writing or interacting.
