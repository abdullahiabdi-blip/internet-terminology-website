📨 HTTP (Hypertext Transfer Protocol)

HTTP is the protocol used for transferring web pages, images, and other resources between web servers and browsers.

How HTTP works:

  • GET request: Browser asks server for a webpage
  • POST request: Browser sends data to server (like form submissions)
  • Response: Server sends back the requested content

HTTP vs HTTPS: HTTPS is the secure version with encryption (the 'S' stands for Secure).

Common HTTP status codes: 200 (OK), 404 (Not Found), 500 (Server Error).
