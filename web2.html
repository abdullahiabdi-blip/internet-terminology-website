📱 Web 2.0 (The Read-Write Web)

Web 2.0 emerged around 2004 and transformed the internet from static pages to interactive platforms.

Characteristics of Web 2.0:

  • Read-Write: Users can both consume AND create content
  • Social Media: Platforms like Facebook, YouTube, Twitter emerge
  • User-generated content: Comments, reviews, posts, videos
  • Interactive: Likes, shares, follows, and discussions
  • Collaboration: Wikis, blogs, and crowdsourcing
  • Dynamic content: AJAX, JavaScript, responsive design

Examples of Web 2.0: Facebook, YouTube, Wikipedia, Instagram, Reddit, TikTok.

Web 2.0 is about participation and interaction - turning users into creators.
