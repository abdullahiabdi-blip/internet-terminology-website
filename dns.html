🌐 DNS (Domain Name Server)

DNS is like the phonebook of the internet. It translates human-friendly domain names (like google.com) into IP addresses (like 142.250.190.46) that computers use to identify each other.

How DNS works:

  1. You type a domain name in your browser
  2. Your computer asks a DNS server for the IP address
  3. DNS server finds the matching IP address
  4. Your computer connects to that IP address
  5. The website loads

Without DNS, we would have to remember long numbers for every website!
