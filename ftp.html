📁 FTP (File Transfer Protocol)

FTP is a standard network protocol used to transfer files between a client and a server on a computer network.

Common FTP uses:

  • Uploading website files to a web server
  • Downloading large files from servers
  • Sharing files between computers
  • Backing up data to remote servers

FTP vs SFTP: SFTP (SSH File Transfer Protocol) is the secure version with encryption.

Popular FTP clients: FileZilla, Cyberduck, WinSCP.

FTP uses port 21 for communication by default.
