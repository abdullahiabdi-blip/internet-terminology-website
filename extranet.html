🤝 Extranet

An extranet is a controlled private network that allows access to partners, vendors, suppliers, or customers - extending beyond the organization's intranet.

Key features:

  • Extends intranet to authorized external users
  • Requires login credentials for access
  • More secure than public internet access
  • Used for business-to-business communication

Examples of extranet use: Supply chain management, customer portals, partner collaboration, vendor access to inventory.

Difference: Intranet = employees only | Extranet = employees + approved outsiders.
